This is YAMT!

21 May 2020 - Less than 1 minute read

Welcome to YAMT - Yet Another Minimal Theme! This is, of course, just another minimal theme for Jekyll, built with speed and ease of customization in mind. In this post I’ll show you the basic features of YAMT and the more advanced ones.

  • Getting started
  • Basic Features
  • Advanced Features

Getting started

Want to start using YAMT right away? Start by reading this Getting Started post.

Basic Features

  • Minimal and clean theme, lightweight and optimized for mobile;
  • Easily customizable;
  • Great code highlighting.

Advanced Features

  • Works out of the box with Netlify’s form handler.
  • MathJAX and LaTeX full support.
